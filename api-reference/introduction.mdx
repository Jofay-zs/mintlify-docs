## Prerequisites

Loopay provides two environments for your integration:

**Environment for development and testing:**

- Dashboard URL: `demo.loopay.com`
- API Base URL: `https://api.loopay.com`

**Production Environment:**

- Dashboard URL: `app.loopay.com`
- API Base URL: `https://v1.api.loopay.com`

## Authentication

To access the API, authenticate by calling the `/login` endpoint with your Loopay platform credentials (email and password). A token will be returned, which must be included in subsequent requests to other endpoints.
